# **App Name**: ExpertMind Research Engine

## Core Features:

- Question Input: Allows users to input research questions, project descriptions, paper topics, datasets, or vague ideas to initiate the expert mindset analysis.
- Persona Generation: Generates a set of expert personas tailored to the input question, providing diverse perspectives like neuroscientist, ML professor, statistician, YC founder, etc.
- Expert Workflow Generation: For each persona, generates a detailed workflow including problem reframing, hypothesis formulation, experimental design, expected failure modes, and prior literature comparison using AI. This feature requires the tool to carefully manage constraints and prioritize knowledge in different domains.
- Parallel Expert Comparison: Builds a meta-summary that identifies agreements, divergences, conflicting methodologies, promising directions, and missed risks across all personas.
- Meta-Learning Panel: Highlights the best thinking types for different phases, beginner lessons, and transferable skills across domains.
- Comparison Dashboard: A three-column layout with an expert rail, a problem workspace, and a parallel expert view to facilitate comparison of expert workflows.
- Modes: Offers Deep-Dive, Compare, Teaching, and Socratic modes for detailed analysis, horizontal comparison, educational insights, and AI-guided questioning.

## Style Guidelines:

- Primary color: Deep blue (#3F51B5) to evoke intellect, depth, and trust.
- Background color: Light blue-gray (#E8EAF6) for a calm, professional feel.
- Accent color: Soft orange (#FFAB40) to highlight key insights and call to actions, contrasting with the blue for visibility.
- Headline font: 'Space Grotesk', a modern sans-serif font with a techy, scientific feel. Body font: 'Inter', a grotesque sans-serif with a neutral, machined look.
- Use clean, minimalist icons representing different expert personas and research phases.
- A structured three-column layout to facilitate comparison of expert perspectives.
- Subtle animations when generating expert workflows and highlighting key insights.
